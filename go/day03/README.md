# Day 3: Battery Bank Joltage

**Problem**: [Advent of Code 2025 - Day 3](https://adventofcode.com/2025/day/3)

## Problem Description

Calculate maximum joltage from battery banks and sum them up.

## Solution Approach

Process battery bank configurations to find maximum joltage values.
## How to Run

```bash
# From go/day03 directory
go run main.go

# Run tests
go test -v

# Using Makefile from project root
make go-run DAY=3
```

## Complexity

- **Time**: O(n)
- **Space**: O(1)
