# Day 4: Paper Roll Accessibility

**Problem**: [Advent of Code 2025 - Day 4](https://adventofcode.com/2025/day/4)

## Problem Description

Count accessible paper rolls and simulate iterative removal.

## Solution Approach

Process paper roll stack configuration to determine accessibility rules.

## How to Run

```bash
# From go/day04 directory
go run main.go

# Run tests
go test -v

# Using Makefile from project root
make go-run DAY=4
```

## Complexity

- **Time**: O(n)
- **Space**: O(n)
